## Instalation 

1. Clone repository
```shell
git clone https://github.com/ServerlessTrIT/practica-final-aws-franvalverde.git 
```
2. Get in the project folder
```shell
cd front
```
3. Install dependencies via npm or yarn
```shell
npm install
```

```shell
yarn
```
